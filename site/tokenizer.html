Limitações desta demo

Esta é uma aproximação educacional usando uma heurística BPE simples — útil para demonstrar como BPE quebra palavras, mas não é o tokenizador real dos modelos citados. Para contagem exata:

  • OpenAI Tokenizer — visualizador oficial GPT (cl100k_base, o200k_base).
  • Tiktokenizer — inclui Claude, Llama, GPT.
  • Anthropic Token Counting API — contagem exata via API.
💡
Observe o comportamento
  • Palavras comuns em inglês → 1 token. "Programação" em português → 2-3 tokens.
  • Código com identificadores em camelCase ou snake_case é quebrado nos limites.
  • Cada espaço/tab/newline costuma consumir ao menos 1 token.
  • A razão chars/token é ~4 em inglês, ~3 em português — traduza este texto e compare.
